**思路：**

爬楼梯就是斐波那契数列

1. **dp[i]含义：** 第 i 个数的值，**i** 含义：第 i 个数  
2. **递推公式** dp[i] = dp[i-1] + dp[i-2] 后一个数是前两个数的和  
3. **初始化** dp[0] = 0, dp[1] = 1; 
4. **遍历顺序** 从前往后遍历

---

## 代码

空间复杂度 O(1)版的，用了数的迭代

```cpp
class Solution {
public:
    int fib(int n) {
        if(n <= 1) return n;

        int dp[2];
        dp[0] = 0;
        dp[1] = 1;
        int f;

        for(int i = 2; i <= n; i++){
            f = dp[0] +dp[1]; // 求下一个数
            dp[0] = dp[1]; // 求再下一个数时，两个相加数要迭代，往后移
            dp[1] = f;
        }

        return f;
    }
};
```
