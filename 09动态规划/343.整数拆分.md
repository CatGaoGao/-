# 343.整数拆分

力扣链接：https://leetcode.cn/problems/integer-break/

---

第一次理解感觉还是有点困难，有点绕

---

**思路：**

1. **dp[i]含义：** 拆分这个数得到的最大乘积，**i** 含义：第 i 个数   
2. **递推公式** dp[i] = max(dp[i], max(j*(i-j), j*dp[i-j]));
3. 从1遍历 j ，比较 (i-j)*j 和 dp[i-j]*j 取最大的； j*(i-j) 是单纯的把整数拆分为两个数相乘，而 j*dp[i-j] 是拆分成两个以及两个以上的个数相乘
4. **初始化** dp[2] = 1，题目要求从2开始
5. **遍历顺序** 从前往后

---

## 代码

```cpp
class Solution {
public:
    int integerBreak(int n) {
        // 定义dp数组
        vector<int> dp(n+1);

        // 初始化
        dp[2] = 1;

        // 遍历，注意循环的条件
        for(int i = 3; i <= n; i++){
            for(int j = 1; j <= i/2; j++){
                // 递推公式
                dp[i] = max(dp[i], max(j*(i-j), j*dp[i-j]));
            }
        }

        return dp[n];
    }
};
```
