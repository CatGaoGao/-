# 977. 有序数组的平方

力扣链接：https://leetcode.cn/problems/squares-of-a-sorted-array/

---

有了前面铺垫的双指针左右边界、快慢指针解法，这题是有思路的，
但是实现的时候也是细节地方写不好，总是报错

---

**思路：**

1. 利用数组已经有的排序，两边的平方后是最大的，倒着填充回来
2. 设定左右边界下标，在if中平方比大小，然后填充给`result`   
3. 注意`result`是倒着填的，从n-1开始 

---

## 双指针（左右指针）

```cpp
class Solution {
public:    //双指针，巧妙用原有的排序，设置左右指针
    vector<int> sortedSquares(vector<int>& nums) {
        int n = nums.size();
        int left = 0, right = n-1;
        vector<int> result(n,0);
        // 初始化result数组，n个元素，初始值为0

        for(int i = n-1; i >= 0; i--){
            if(nums[left] * nums[left] < nums[right] * nums[right]){
                result[i] = nums[right] * nums[right];
                right--;
            }
            else{
                result[i] = nums[left] * nums[left];
                left++;
            }
        }

        return result;
    }
};
```
