# 59.螺旋矩阵II

力扣链接：https://leetcode.cn/problems/spiral-matrix-ii/

---

但有的就是啥思路没有，还是写了题有总结好一些，上学期期末前做的那些写了总结的至少都是记得思路的

**思路：**

一圈一圈的循环、左闭右开，注意初始设置：开始坐标、循环次数、边界收缩、中心坐标值（奇数时）、赋值数

1. while循环loop次数，初始化坐标   
2. for循环四条边，左闭右开   
3. 更新x、y坐标，更新边界收缩值
4. 若为奇数，单独赋值中心点

---

## 代码

```cpp
class Solution {
public:
    vector<vector<int>> generateMatrix(int n) {
        // 创建答案容器,并初始化
        vector<vector<int>> result(n, vector<int> (n, 0));
        // 创建行列开始的坐标
        int x = 0, y = 0;
        // 循环次数
        int loop = n/2;
        // 边界收缩
        int offb = 1;
        // 若为奇数则会有中心值
        int mid = n/2;
        // 赋值给每一个框,一直到n^2
        int count = 1;

        // 开始写循环，一圈一圈的进行填充
        while(loop--){
            int i = x; // 作为行
            int j = y; // 作为列

            for(j; j < n - offb; j++){
                // 左闭右开，留下一行的最后一个元素作为下一边的开头
                result[i][j] = count;
                count++;
            }

            for(i; i < n - offb; i++){
                // 左闭右开，留下一行的最后一个元素作为下一边的开头
                result[i][j] = count;
                count++;
            }

            for(j; j > x; j--){
                // 左闭右开，留下一行的最后一个元素作为下一边的开头
                result[i][j] = count;
                count++;
            }

            for(i; i > y; i--){
                // 左闭右开，留下一行的最后一个元素作为下一边的开头
                result[i][j] = count;
                count++;
            }

            x++;
            y++; // 第二次循环从（1,1）开始

            offb++; // 第二次循环边界要收缩进去1
        }

        // 如果n为奇数，要单独赋值中心点
        if(n%2 == 1){
            result[mid][mid] = n*n;
        }

        return result;
    }
};
```
